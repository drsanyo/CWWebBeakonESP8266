Frequency range: 1500 kHz - 7500 kHz.
RF output power: 5W - 2W (depending on frequency).



------------- Краткие пояснения к принципиальной схеме и работе маяка (rus) -----------------


 - Резистор R1 установлен для уменьшения мощности, выделяемой на стабилизаторе 7805. Можно этот резистор не устанавливать,
тогда 7805 будет греться немного больше.

 - Реле применены РГК-15. Можно применить другие с двумя группами на замыкание или на переключение, или по два отдельных
реле с одной группой.

 - Диод Д5 для защиты от переполюсовки.

 - Цепь R2, R3, D1 предназначена для смещения вверх дипазона регулируемых постоянных напряжений, получаемых с ШИМ, 
так как напряжения +3В не достаточно для получения такого тока стока транзистора VT1, при котором развивается максимальная мощность.

 - Цепь R4, C5 предназначена для подавления частоты 10 кГц, на которой работает ШИМ.

 - В качестве транзистора VT1 можно применять только оригинальный IRF510. В продаже есть много подделок с маркировкой IRF510, 
но работающих только в ключевом режиме. Например на Aliexpress такие продаются у многих продавцов. Такие транзисторы не работают
в усилителях ВЧ. Оригинальный IRF510 позволяет плавно регулировать ток покоя.
Конечно же можно применить и другие типы полевых транзисторв, подходящие по параметрам.

 - Частоты среза ФНЧ 2500 кГц, 4300 кГц и 7500 кГц выбраны для перекрытия всего диапазона частот от 1500 до 7500 кГц.
   Если предполагается работа маяка только в одном диапазоне частот, например 80 м или 40 м, то схему коммутации ФНЧ 
можно не устанавливать, а установить только один требуемый ФНЧ.

 - Ферритовые кольца для катушек ФНЧ  К7х4х2 проницаемость 50 ВЧ. Возможно использовать друге кольца, рассчитав количество
витков для получения требуемой индуктивности.

 - Дроссель DR1 можно изготовть на кольце 2000НН или использовать готовый дроссель типа ДМ.

 - При включении программа маяка выдаёт в текстовом виде отладочную информацию на последовательный порт, который доступен через USB.
В отладочную информацию входит текстовый дамп настроек маяка, результаты запуска WiFi и т.п..

 - В режиме "точка доступа" маяк видно по WiFi с SSID "RadioBeacon". При подключении к нему как к "точке доступа" он выдаёт
клиенту по DHCP адрес IP из подсети 192.168.55.0/24. Сам маяк доступен по IP 192.168.55.1

 - Для работы на частотах выше 7.5...10 МГц кроме других ФНЧ нужно еще использовать другой транзистор в усилителе мощности.
Транзистор должен иметь меньгую ёмкость затвора. Или можно использовать другую схему усилителя мощности.
